A PyGame implementation of a classic Memory card game written in a few hours for a coding challenge. 

External libraries:

    PyGame

Instructions to run:
    Clone this repo from the command line:
        git clone git@github.com:JacquelineMai/pygameMemory.git with ssh or
        git clone https://github.com/JacquelineMai/pygameMemory.git with https
    If you do not already have PyGame installed, run pip install pygame
    Enter the cloned directory and launch the game with python3 game.py

Instructions to play:
    Click on a card to reveal its face. Click on another one to match the exposed card. If they are a pair, they will both remain face-up.
